# 3d triamgles readme

## project launch

1. copying project
```
git clone https://github.com/2APetrin/3d_triangles.git
cd 3d_triangles
```

2. making
```
cmake -B build
cd build
make
```

3. launching
main prog:
```
./source/triag
```

unit tests:
```
./tests/test_main
```

4. end-to-end tests: </br>
*from 3d_triandles directory:*
```
./test/tests.sh
```
